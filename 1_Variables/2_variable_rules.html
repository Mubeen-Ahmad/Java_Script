Rules of Variables

Use the following rules when you name procedures, constants, variables, and arguments

Rule 1

Letters (a,b,c)
Digits (1,2,3)
underscore ( _ )
$ Sign are Allowed

name = "Mubeen";
no1 = 123;
no_2 = 345;
$_no3 = 678

document.write(name);
document.write(no1);
document.write(no2);
document.write($_name);

Rule 2

Must be begain with :-
( $ ),
( _ ) ,
or ( letters )

name = "Mubeen";
$12 = 123;
_23 = 345;

document.write(name);
document.write($12);
document.write(_23);

Rule 3

Reserved word are not used as a variable name
var = "Mubeen";
let = 123

document.write(var);
document.write(let);

Rule 4

A and a are Two Different Variables
its Case Sensitive

A = "Mubeen";
a = "Ahmad"

document.write(A);
document.write(a);
